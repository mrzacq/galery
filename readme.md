# How to run

# server
```
Requirement: {
    postgres, npm
}
```
Run
```
1. npm install
2. configure config.json in config folder. set password to your postgres password
3. npx sequelize-cli db:create
4. npx sequleize-cli db:migrate
5. npm run dev
```

# client

```
Requirement: {
    vue, vue-cli, npm
}
```
Run
```
1. npm install
2. npm run serve
```
